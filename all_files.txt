                        SCHEMA DISTRIBUTION
================================================================================

PUBLIC SCHEMA           TENANT SCHEMAS              NOTES
────────────────────────────────────────────────────────────────────────────────
tenants                 roles                      Tenant creates own schema via
permissions            users                      FlywayMigrationService on signup
plans                   refresh_tokens
platform_admins         role_permissions
tenant_onboarding       customers                  [V15]
                         products                   [V16]
                         subscriptions              business_invoices              [V17]
                         usage_records              business_invoice_items         [V18]
                         usage_summaries           business_payments              [V19]
                         invoices
                         invoice_line_items
                         payments
                         QRTZ_* (Quartz tables)

CROSS-SCHEMA NOTES:
  - tenant.schema_name       -> PostgreSQL schema name (one per tenant)
  - subscriptions.plan_id    -> public.plans.id (denormalized, no FK)
  - role_permissions.permission_id -> public.permissions.id (via CROSS JOIN)
  - tenant_onboarding.tenant_id -> public.tenants.id (logical FK only)

================================================================================
                         PERMISSIONS (10 TOTAL)
================================================================================

Name              Description
────────────────────────────────────────────────────────────────────────────────
TENANT_VIEW       View tenant details and settings
TENANT_MANAGE     Update tenant settings and configurations
USER_VIEW         View users within the tenant
USER_MANAGE       Create and update users within the tenant
USER_DELETE       Delete users from the tenant
ROLE_VIEW         View roles and their permissions
ROLE_MANAGE       Create, update and assign roles
BILLING_MANAGE    Manage subscriptions, invoices and payments
CUSTOMER_MANAGE   Create, view, update and delete customers
PRODUCT_MANAGE    Create, view, update and deactivate products
